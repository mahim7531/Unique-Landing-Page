# WorkHive – Freelance Marketplace Landing Page

A modern and responsive **Upwork-style freelance marketplace landing page** built using **HTML and CSS**.
This project demonstrates a clean UI layout suitable for freelance platforms, job marketplaces, or SaaS products.

---

## 🚀 Features

* Modern Upwork-inspired UI
* Responsive hero section
* Navigation bar with login/signup buttons
* Popular services section
* Freelancer profile cards
* Clean and minimal footer
* Simple and beginner-friendly code structure

---

## 🛠 Technologies Used

* HTML5
* CSS3
* Flexbox
* CSS Grid
